<h2>📌 Project Overview</h2>

<p>
The AI Chatbot is a web-based application developed using <b>Python</b>,
<b>Flask</b>, <b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b>.
The chatbot interacts with users through an attractive interface,
understands user queries, and provides intelligent responses based on
predefined intents stored in a JSON file.
</p>

<p>
This project demonstrates the fundamentals of Artificial Intelligence,
Natural Language Processing (NLP), and Web Development.
</p>

<h2>⚙️ Installation</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone &lt;repository-url&gt;
cd AI_Chatbot
</pre>

<h3>2️⃣ Create Virtual Environment</h3>

<pre>
python -m venv venv
</pre>

<h3>3️⃣ Activate Virtual Environment</h3>

<b>Windows</b>

<pre>
venv\Scripts\activate
</pre>

<b>Linux / Mac</b>

<pre>
source venv/bin/activate
</pre>

<h3>4️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

or

<pre>
pip install flask
</pre>

<hr>

<h2>▶️ Run the Application</h2>

<pre>
python app.py
</pre>

<p>Open your browser and visit:</p>

<pre>
http://127.0.0.1:5000
</pre>

<h2>🧠 How It Works</h2>

<ol>
<li>User enters a question.</li>
<li>Flask receives the request.</li>
<li>chatbot.py processes the input.</li>
<li>User input is compared with patterns in responses.json.</li>
<li>Best matching intent is identified.</li>
<li>A random response from the matched intent is returned.</li>
<li>Response is displayed dynamically in the chat interface.</li>
</ol>
